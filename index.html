Technical Projects

License Plate Recognition System (OCR)

A prototype system for automatic license plate detection and recognition using computer vision techniques. The system processes video input, isolates the plate region, applies image preprocessing and extracts the alphanumeric text via Optical Character Recognition. The project is currently experimental and under continuous improvement.

Video Input → Plate Detection → Preprocessing → OCR Recognition → Text Output
