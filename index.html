Showcase

Progetto
Principale

License Plate
Recognition System

Sistema di riconoscimento automatico delle targhe automobilistiche sviluppato in Python. Il programma acquisisce immagini o feed video, individua la targa, ne estrae il testo tramite OCR e lo registra in un database locale.

// La Sfida Tecnica

Il problema principale era la variabilità delle condizioni di illuminazione e angolazioni delle targhe. L'OCR grezzo produceva output errati fino al 40% dei casi in condizioni non ideali.

// La Soluzione

Pipeline di pre-processing con OpenCV: grayscale → adaptive thresholding → morphological operations → contour detection. Accuratezza finale >85% in condizioni reali.

Python 3.11 OpenCV Tesseract OCR NumPy SQLite
