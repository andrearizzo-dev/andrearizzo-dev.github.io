Technical
Projects

License Plate
Recognition System

A prototype system for automatic license plate detection and recognition using computer vision. The system processes video input, isolates the plate region, applies image preprocessing and extracts alphanumeric text via OCR.

Video Input → Detection → Preprocessing → OCR → Text Output

// The Technical Challenge

Variable lighting conditions and plate angles caused raw OCR to misread up to 40 % of frames in non-ideal conditions.

// The Solution

OpenCV preprocessing pipeline: grayscale → adaptive thresholding → morphological operations → contour detection. Final accuracy > 85 % in real conditions.

Python OpenCV OCR NumPy Experimental
